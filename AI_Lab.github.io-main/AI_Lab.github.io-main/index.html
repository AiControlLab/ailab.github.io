飞艇

专注于提升高空柔性飞艇的整体性能，主要涵盖模态分析、气动特性研究和推进系统优化三个方面。通过结构动力学分析与相似准则，结合实验与仿真验证，系统研究飞艇在不同工况下的固有频率和振动模式，为模态预测提供可靠理论支持。在气动特性方面，深入分析气流与柔性结构的耦合作用，优化设计以提升飞行效率与稳定性。针对动力系统，聚焦提升推进效率与控制精度，解决现有动力不足和响应迟缓等瓶颈。项目旨在多维度提升柔性飞艇的性能，为其在空中运输、气象监测和军事侦察等领域的应用奠定基础

支吊架智能巡检系统

支吊架智能巡检系统

流固耦合飞艇颤振主动抑制

飞艇的运动和振动问题通常涉及复杂的非线性行为。扰动分析图通过线性化方法将问题简化，帮助工程师理解系统在小扰动下的响应，从而避免对全系统复杂非线性分析的过度依赖。并且扰动分析图能够帮助分析飞艇在不同条件下的模态特性，揭示系统的固有频率和振型。通过对扰动的响应进行分析，可以识别出可能影响飞艇稳定性的关键模态，进而指导设计和优化。通过扰动分析，飞艇的动态特性可以被量化，并且不同行为模式对飞行性能的影响可以得到评估。这样的分析为控制系统设计提供了理论依据，帮助设计更加稳定和高效的飞行控制系统。
